This website is a personal portfolio built for the CS50 Homepage assignment. It includes four pages: index.html, about.html, projects.html,
and contact.html. Each page uses a shared sidebar for navigation and a consistent layout created with Bootstrap and custom CSS.

HTML Tags Used:
The project uses more than ten distinct HTML tags, including:
nav, ul, li, a, div, h1, h2, h5, p, img, button, footer, hr, script, link, strong

CSS Selectors Used:
The custom stylesheet includes multiple selectors, such as:
.sidebar, .hero-text, .main-container, .profile-pic, .project-card, .project-card-usellit, .project-card-spacecowboy, .nav-link:hover

CSS Properties Used:
The project uses more than five CSS properties, including:
background-image, background-size, background-position, background-repeat, border-radius, padding, margin, color, display, position, overflow,
box-shadow, backdrop-filter

Bootstrap Usage:
Bootstrap 5.3.8 is used throughout the site for layout and styling. Components include the grid system, cards, buttons, modals, spacing utilities,
and responsive classes. Bootstrap’s modal system is used to display project details.

JavaScript Usage:
JavaScript is used to add interactivity to the site. A function on the homepage triggers a welcome modal. On the projects page, JavaScript dynamically
loads project screenshots into modals, displays loading spinners, and manipulates the DOM when modals are opened.

Additional Notes:
All pages share a fixed sidebar for navigation. The main content area uses a full-page background image with a semi-transparent hero section for
readability. Images used in modals are loaded from the images/ directory.
